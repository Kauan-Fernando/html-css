outros tipos de formatações

fontes mono espaçadas

para termos fonte monoespaçadas usamos a tag code usamos para codigos

se o codigo tiver mais de uma linha colocamos o code dentro da tag pre

citações

para alguma citação usamo a tag qisso se for uma citaçao pquena

se for uma citacao grande usamos a tag blockquote

abreviações

para mostrar oque e o significado se uma sigla usamos a sigla abbr
